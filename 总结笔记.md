# 18 Four Sum #Twopointer

# 19 Remove nth Node From End of List #Twopointer

本题主要要通过一次遍历来完成删除, 只需要设定两个pointer, 一个先开始遍历, 另外一个慢一点, 两个pointer之间正好相隔n个距离, 这样当快pointer到达尾部, 慢pointer也正好到达需要删除的位置, 只需通过一边遍历就可完成删除
